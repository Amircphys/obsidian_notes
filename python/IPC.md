### **3. –ö–∞–Ω–∞–ª—ã (Pipes): –æ–±–º–µ–Ω –¥–∞–Ω–Ω—ã–º–∏ –º–µ–∂–¥—É –ø—Ä–æ—Ü–µ—Å—Å–∞–º–∏**
**–ó–∞—á–µ–º:** –ö–æ–≥–¥–∞ –¥–≤—É–º –ø—Ä–æ—Ü–µ—Å—Å–∞–º –Ω—É–∂–Ω–æ –æ–±–º–µ–Ω–∏–≤–∞—Ç—å—Å—è –ø—Ä–æ—Å—Ç—ã–º–∏ —Å–æ–æ–±—â–µ–Ω–∏—è–º–∏ (–Ω–∞–ø—Ä–∏–º–µ—Ä, —Ä–æ–¥–∏—Ç–µ–ª—å—Å–∫–∏–π –∏ –¥–æ—á–µ—Ä–Ω–∏–π –ø—Ä–æ—Ü–µ—Å—Å).  
**–ö–∞–∫:** –ß–µ—Ä–µ–∑ `os.pipe()` –∏–ª–∏ `multiprocessing.Pipe`.

**–ü—Ä–∏–º–µ—Ä —Å `multiprocessing`:**

```python
    print(f"–†–µ–±–µ–Ω–æ–∫ –ø–æ–ª—É—á–∏–ª: {msg}")

# –°–æ–∑–¥–∞—Ç—å –∫–∞–Ω–∞–ª
—Ä–æ–¥–∏—Ç–µ–ª—å_conn, —Ä–µ–±–µ–Ω–æ–∫_conn = Pipe()

# –†–æ–¥–∏—Ç–µ–ª—å –ø–æ–ª—É—á–∞–µ—Ç —Å–æ–æ–±—â–µ–Ω–∏–µ
print(—Ä–æ–¥–∏—Ç–µ–ª—å_conn.recv())  # –í—ã–≤–æ–¥: –ü—Ä–∏–≤–µ—Ç –æ—Ç —Ä–µ–±–µ–Ω–∫–∞!

# –†–æ–¥–∏—Ç–µ–ª—å –æ—Ç–ø—Ä–∞–≤–ª—è–µ—Ç –æ—Ç–≤–µ—Ç
—Ä–æ–¥–∏—Ç–µ–ª—å_conn.send("–ü—Ä–∏–≤–µ—Ç –æ—Ç —Ä–æ–¥–∏—Ç–µ–ª—è!")
p.join()
```

### **4. mmap**
**mmap** ‚Äî —ç—Ç–æ —Å–ø–æ—Å–æ–± —Ä–∞–±–æ—Ç–∞—Ç—å —Å —Ñ–∞–π–ª–æ–º, –∫–∞–∫ —Å –∫—É—Å–∫–æ–º –æ–ø–µ—Ä–∞—Ç–∏–≤–Ω–æ–π –ø–∞–º—è—Ç–∏.  
–ü—Ä–µ–¥—Å—Ç–∞–≤—å—Ç–µ, —á—Ç–æ —Ñ–∞–π–ª –Ω–∞ –¥–∏—Å–∫–µ —Å—Ç–∞–Ω–æ–≤–∏—Ç—Å—è —á–∞—Å—Ç—å—é –ø–∞–º—è—Ç–∏ –≤–∞—à–µ–π –ø—Ä–æ–≥—Ä–∞–º–º—ã. –í—ã –º–æ–∂–µ—Ç–µ —á–∏—Ç–∞—Ç—å –∏ –∏–∑–º–µ–Ω—è—Ç—å –µ–≥–æ –¥–∞–Ω–Ω—ã–µ —Ç–∞–∫ –∂–µ –±—ã—Å—Ç—Ä–æ, –∫–∞–∫ –æ–±—ã—á–Ω—ã–µ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–µ, –±–µ–∑ –ø–æ—Å—Ç–æ—è–Ω–Ω—ã—Ö –æ–ø–µ—Ä–∞—Ü–∏–π —á—Ç–µ–Ω–∏—è/–∑–∞–ø–∏—Å–∏ –Ω–∞ –¥–∏—Å–∫.

**–ó–∞—á–µ–º —ç—Ç–æ –Ω—É–∂–Ω–æ?**  
- **–°–∫–æ—Ä–æ—Å—Ç—å:** –†–∞–±–æ—Ç–∞ —Å –ø–∞–º—è—Ç—å—é –±—ã—Å—Ç—Ä–µ–µ, —á–µ–º —Å –¥–∏—Å–∫–æ–º.

- **–ë–æ–ª—å—à–∏–µ —Ñ–∞–π–ª—ã:** –£–¥–æ–±–Ω–æ –¥–ª—è —Ä–∞–±–æ—Ç—ã —Å –æ–≥—Ä–æ–º–Ω—ã–º–∏ —Ñ–∞–π–ª–∞–º–∏ (–Ω–∞–ø—Ä–∏–º–µ—Ä, –≤–∏–¥–µ–æ), —Ç–∞–∫ –∫–∞–∫ —Ñ–∞–π–ª –Ω–µ –∑–∞–≥—Ä—É–∂–∞–µ—Ç—Å—è –≤ –ø–∞–º—è—Ç—å —Ü–µ–ª–∏–∫–æ–º.

### **–ü—Ä–∏–º–µ—Ä 1: –ë–∞–∑–æ–≤–æ–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ mmap**
```python
import mmap
import os

# –®–∞–≥ 1: –°–æ–∑–¥–∞–µ–º —Ñ–∞–π–ª –∏ –∑–∞–ø–æ–ª–Ω—è–µ–º –µ–≥–æ –Ω—É–ª—è–º–∏

    file.write(b"\x00" * 1024)  # –§–∞–π–ª —Ä–∞–∑–º–µ—Ä–æ–º 1 –ö–ë

# –®–∞–≥ 2: –û—Ç–∫—Ä—ã–≤–∞–µ–º —Ñ–∞–π–ª –¥–ª—è —á—Ç–µ–Ω–∏—è/–∑–∞–ø–∏—Å–∏ –∏ –æ—Ç–æ–±—Ä–∞–∂–∞–µ–º –µ–≥–æ –≤ –ø–∞–º—è—Ç—å
with open("demo.bin", "r+b") as file:
    # –°–æ–∑–¥–∞–µ–º mmap-–æ–±—ä–µ–∫—Ç
    mm = mmap.mmap(file.fileno(), 0)  # 0 ‚Äî –æ—Ç–æ–±—Ä–∞–∑–∏—Ç—å –≤–µ—Å—å —Ñ–∞–π–ª

    # –®–∞–≥ 3: –ó–∞–ø–∏—Å—ã–≤–∞–µ–º –¥–∞–Ω–Ω—ã–µ –≤ –ø–∞–º—è—Ç—å
    mm[0:5] = b"Hello"  # –ó–∞–ø–∏—Å—ã–≤–∞–µ–º "Hello" –≤ –Ω–∞—á–∞–ª–æ —Ñ–∞–π–ª–∞

    # –®–∞–≥ 4: –ß–∏—Ç–∞–µ–º –¥–∞–Ω–Ω—ã–µ
    mm.seek(0)  # –ü–µ—Ä–µ–º–µ—â–∞–µ–º—Å—è –≤ –Ω–∞—á–∞–ª–æ

    # –ó–∞–∫—Ä—ã–≤–∞–µ–º mmap –∏ —Ñ–∞–π–ª (–≤–∞–∂–Ω–æ!)
    mm.close()
```

**–ü–æ—è—Å–Ω–µ–Ω–∏—è:**

- `mm[0:5]` ‚Äî —Ä–∞–±–æ—Ç–∞–µ–º —Å –¥–∞–Ω–Ω—ã–º–∏ –∫–∞–∫ —Å –º–∞—Å—Å–∏–≤–æ–º –±–∞–π—Ç–æ–≤.  
- `seek(0)` ‚Äî –ø–µ—Ä–µ–º–µ—â–∞–µ–º—Å—è –≤ –Ω–∞—á–∞–ª–æ —Ñ–∞–π–ª–∞.  
- `mm.close()` ‚Äî –æ–±—è–∑–∞—Ç–µ–ª—å–Ω–æ –∑–∞–∫—Ä—ã–≤–∞–µ–º mmap, –∏–Ω–∞—á–µ –≤–æ–∑–º–æ–∂–Ω—ã –æ—à–∏–±–∫–∏.
